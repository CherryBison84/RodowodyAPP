# Zubry Pedigree App

Aplikacja do analizy rodowodowej żubrów (Tkinter + Streamlit).

## Uruchomienie (po instalacji)

### Tkinter
```bash
cd zubry_pedigree_app
.venv/bin/python -m app.ui.tk.main
```

### Streamlit
```bash
cd zubry_pedigree_app
.venv/bin/streamlit run app/ui/streamlit/streamlit_app.py
```
